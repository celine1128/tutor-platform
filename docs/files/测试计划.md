# 家教管理平台 - 测试计划
## 1. 测试项目
家教管理平台（tutor‑platform）课程设计项目

## 2. 测试目的
验证系统全部功能可用、业务流程正确、权限控制有效、无明显BUG，满足课程设计交付要求。

## 3. 测试范围
1. 用户模块：注册、登录、退出、密码加密
2. 权限模块：未登录拦截、角色越权拦截
3. 家长模块：需求发布、我的需求、订单确认
4. 老师模块：需求浏览、接单、我的订单
5. 数据流转：需求、订单状态变更

## 4. 测试环境
- 操作系统：Windows 10/11
- JDK：1.8
- 数据库：MySQL 8.0
- 浏览器：Google Chrome
- 运行地址：http://localhost:8080/user/login

## 5. 测试方法
采用**黑盒功能测试**，不关注代码实现，仅验证功能输入输出、页面跳转、数据变更是否符合预期。

## 6. 测试时间
1 天内完成全量功能测试

## 7. 测试人员
项目开发人员
