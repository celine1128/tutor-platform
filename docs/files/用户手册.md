# 家教管理平台 - 用户操作手册
## 1. 系统简介
本系统为家长、大学生老师提供家教线上对接服务，实现需求发布、在线接单、订单确认全流程线上管理。

## 2. 运行环境与访问地址
- 运行环境：JDK1.8、MySQL8.0、Chrome浏览器
- 本地访问地址：http://localhost:8080/user/login

## 3. 账号注册与登录
### 3.1 注册账号
1. 进入登录页，点击【账号注册】
2. 输入手机号、登录密码，选择身份：**家长 / 大学生老师**
3. 提交注册，自动跳转登录页面

### 3.2 登录系统
1. 输入注册手机号、密码
2. 系统自动识别角色，跳转对应功能首页

## 4. 家长端操作指南
1. **发布家教需求**：首页点击【发布家教需求】，填写辅导科目、年级、时间、地点、期望时薪，提交发布
2. **查看我的需求**：点击【我的需求】，查看本人发布的所有家教需求，可查看待接单/已接单状态
3. **确认订单完成**：老师接单后生成订单，进入【我的订单】，辅导结束后点击【确认完成】
4. **退出登录**：点击【退出登录】，返回登录页面

## 5. 老师端操作指南
1. **查看待接单需求**：首页点击【查看待接需求】，浏览所有家长发布的未接单家教需求
2. **在线接单**：选择合适需求，点击【立即接单】，接单成功后自动生成订单
3. **查看我的订单**：点击【我的接单订单】，查看本人所有接单订单，区分待确认/已完成
4. **退出登录**：点击【退出登录】，返回登录页面

## 6. 注意事项
1. 仅**待接单**的需求可接单，已接单需求无法重复操作
2. 仅**待确认**的订单可确认完成，已完成订单不可修改
3. 不同角色账号无法访问对方专属功能页面
