# 家教管理平台 - 测试用例
| 用例ID | 测试模块 | 测试步骤 | 预期结果 |
| :---: | :---: | :--- | :--- |
| U001 | 用户注册 | 输入手机号、密码、选择角色，点击注册 | 注册成功，密码MD5加密，跳转登录页 |
| U002 | 用户登录 | 输入正确手机号、密码，点击登录 | 登录成功，按角色跳转对应首页 |
| U003 | 登录校验 | 输入错误密码登录 | 提示账号密码错误，停留在登录页 |
| U004 | 权限拦截 | 未登录直接访问家长首页 | 强制跳转至登录页 |
| U005 | 角色越权 | 老师账号访问家长页面 | 跳转登录页，禁止访问 |
| P001 | 发布需求 | 家长登录，填写科目、年级、时间、地点、薪资，提交 | 需求保存成功，状态为待接单，跳转我的需求 |
| P002 | 查看我的需求 | 家长查看个人需求 | 展示本人发布的所有需求，状态正确 |
| P003 | 确认订单 | 家长进入我的订单，点击确认完成 | 订单状态变更为已完成 |
| T001 | 浏览需求 | 老师登录，查看待接单需求 | 仅展示状态为待接单的需求 |
| T002 | 接单操作 | 老师对未接单需求点击接单 | 需求状态变为已接单，自动生成订单 |
| T003 | 重复接单 | 老师对已接单需求再次操作 | 提示无效，无法接单 |
| T004 | 查看我的订单 | 老师查看个人订单 | 展示本人接单的所有订单，状态正确 |
