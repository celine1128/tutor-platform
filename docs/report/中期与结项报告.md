## 一、项目概述
### 1. 项目背景
随着校园家教与社区家教需求不断增加，传统线下对接方式存在信息分散、沟通成本高、订单管理不规范等问题。为提高家教供需双方对接效率，实现需求发布、老师接单、订单跟踪、用户管理等全流程线上化，特开发本家教管理平台。系统面向家长与老师两类用户，提供简洁易用的操作界面与稳定可靠的业务流程，适用于课程实践与小型家教服务管理场景。

### 2. 系统目标
- 实现家长端、老师端完整业务闭环，支持需求发布、老师筛选、接单、订单状态管理等核心功能。
- 保证用户信息安全，实现密码加密、登录状态管理、权限控制与防越权访问。
- 完成系统UI统一美化，提升页面交互体验与兼容性。
- 支持多环境部署与版本管理，代码结构清晰，可运行、可扩展、可维护。
- 完成完整测试、文档与交付流程，满足课程结项要求。

### 3. 开发环境
- 开发工具：IntelliJ IDEA、VS Code、Navicat、Git
- 后端技术：SpringBoot 3.x、MyBatis-Plus、Maven
- 前端技术：Thymeleaf、Bootstrap、Vue 相关组件
- 数据库：MySQL 8.0
- JDK 版本：JDK 17
- 代码托管：GitHub https://github.com/celine1128/tutor-platform.git

## 二、需求分析
### 1. 功能需求
- **用户模块**：注册、登录、找回密码、Session 管理、密码加密存储。
- **家长模块**：发布家教需求、查看我的需求、老师筛选、个人统计看板、订单管理与确认完成。
- **老师模块**：查看可接单需求、接单处理、我的订单列表、订单状态展示与时间显示。
- **订单流程**：需求待接单 → 老师接单 → 订单待确认 → 家长确认完成。
- **系统管理**：权限拦截、版本合并、冗余清理、数据库冲突修复、前后端兼容适配。

### 2. 非功能需求
- **安全性**：用户密码加密，登录校验，角色隔离，防止重复接单与重复确认。
- **兼容性**：支持 JDK 17 + SpringBoot 3.x，主流浏览器均可正常访问。
- **易用性**：界面简洁、流程清晰、操作步骤少，降低用户使用成本。
- **稳定性**：业务流程闭环，数据持久化正常，无明显崩溃与异常。

## 三、系统设计
### 1. 系统架构
采用经典 MVC 三层架构：
- 表现层：Controller 接收请求、页面跳转、参数校验。
- 业务层：Service 实现业务逻辑、状态控制、权限判断。
- 数据层：Mapper + MySQL 实现数据增删改查。
- 前端： Thymeleaf 模板渲染 + Bootstrap 页面美化。

![系统架构图](./assets/fig_architecture.png)

### 2. 模块设计
- 用户模块：注册、登录、找回密码、Session、权限拦截。
- 家长模块：需求发布、我的需求、订单查看、订单确认、统计看板。
- 老师模块：需求大厅、接单、我的订单、订单状态处理。
- 前端美化模块：全站 UI 统一、样式美化、路径修复、渲染优化。
- 测试与部署模块：功能测试、BUG 修复、版本整合、项目部署。

### 3. 数据库设计
#### E-R 图
![E-R 图](./docs/ER/tutor_platform_er.png)

#### 主要数据表
- `sys_user`：用户信息（手机号、密码、角色：家长/老师）
- `tutor_demand`：家教需求（科目、年级、时间、地点、薪资、状态）
- `tutor_order`：订单信息（需求ID、家长ID、老师ID、订单状态）

```java
public class PasswordEncoderUtil {
    public static String encode(String password) {
        return new BCryptPasswordEncoder().encode(password);
    }
    public static boolean matches(String raw, String encoded) {
        return new BCryptPasswordEncoder().matches(raw, encoded);
    }
}
```

#### 登录拦截配置
```java
@Override
public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) throws Exception {
    Object user = request.getSession().getAttribute("user");
    if (user == null) {
        response.sendRedirect("/user/login");
        return false;
    }
    return true;
}
```

## 六、用户手册
### 1. 安装部署
1. 克隆项目：`git clone https://github.com/celine1128/tutor-platform.git`
2. 创建数据库，执行`db.sql`脚本
3. 修改`application.yml`数据库账号密码
4. 使用 IDEA 打开项目，启动启动类
5. 浏览器访问：`http://localhost:8080/user/login`

### 2. 操作指南
- **家长**：注册登录 → 发布家教需求 → 查看我的需求 → 老师接单后查看订单 → 确认完成。
- **老师**：注册登录 → 查看待接单需求 → 接单 → 查看我的订单 → 等待家长确认。
- **公共**：支持找回密码、个人看板、统计信息、筛选老师等功能。
