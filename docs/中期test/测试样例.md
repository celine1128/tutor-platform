### **三、 典型错误（Bug）记录与分析**

#### **1. 家长端“我的订单” 500 错误**

* **现象描述**：点击导航栏“我的订单”时跳转至错误页面（见截图 `022303...`）。
* **初步分析**：后端 SQL 查询关联 `parent_id` 时可能出现字段名不匹配或空指针。
* **证据图片**：`![500报错](详见./assets/.png)`

#### **2. 老师端“需求大厅”为空**

* **现象描述**：数据库中有待接单数据，但老师登录后大厅不显示（见截图 `8835d6...`）。
* **初步分析**：后端 `DemandMapper` 的查询过滤逻辑可能有误。
* **证据图片**：`![大厅为空](详见./assets/.png)`

#### **3. 老师端“立即接单” 404 错误**

* **现象描述**：在大厅点击按钮，浏览器提示路径找不到（见截图 `b5b5a9...`）。
* **初步分析**：前端 Thymeleaf 拼接的 URL 与 Controller 定义的路径不一致。
* **证据图片**：`![404报错](详见./assets/.png)`
